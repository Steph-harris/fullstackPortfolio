Site Scraper

Summary

This site uses the cheerio node package to scrape article links and headlines from mlb.com and save them into a Mongo database. The user can then attach notes to a specific article or delete notes.

Up to date MLB standing are pulled from the xmlstats API. Users can also click on a game to see the home team's venue information and picture, pulled from the FourSquare API.

I used the gameday-helper package to get MLB game data, and HandlebarsJS as my templating engine.

Technologies Used

  • Node.js
  • Express
  • MongoDB
  • HTML5/CSS3
  • Foundation
  • Javascript
  • HandlebarsJS

APIs Used

  • FourSquare
  • Xmlstats

Kata Dashboard

Summary

I often use Codewars to practice coding in languages I haven't used in awhile. In order to quickly see the challenges and languages I've trained in, I created this dashboard in Python that pulls from the Codewars API.

Visitors can search for any valid user and see their solved kata displayed in the Datatable. They can then sort, filter, or page through the results. Clicking on a challenge name will show the kata description and provide a link to the challenge on Codewars.

Technologies Used

  • Python
  • Tornado
  • HTML5/CSS3
  • Bootstrap
  • Javascript/jQuery

APIs Used

  • Codewars
  • DataTables

Show Us a Pet

Summary

This is a shelter pet app that displays pets up for adoption close to a users zip code with images, pet description, special needs, wikipedia article (if a specific breed is provided), and care information pulled from the Flickr, WikiPedia, and PetFinder API's.

Users can also leave a review of their pet via Firebase chat.

Rock, Paper, Scissors

Summary

My version of the popular game featuring Justin Aguilar's CSS3 animations. Users can play any odd number of rounds from 3-9, and any round ending in a tie will be replayed so that there is always a winner. I used Firebase to build a realtime Leaderboard for scores.

  • Javascript
  • jQuery
  • Firebase Leaderboard
