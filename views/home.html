About Me

I’m Stephanie Harris and I’ve recently left a career as a church music director to become a web developer. I actually built my first website when I was 11, but it wasn’t until last year, when I signed up for an online coding bootcamp, that I seriously considered a career in tech. I knew that I would benefit more, and progress quicker, by participating in the more collaborative and immersive experience that an in person bootcamp like Rutgers’ offers. So I switched to Rutgers’ Coding Bootcamp where I’ve developed skills in front-end technologies like HTML, CSS, and Javascript, frameworks like Bootstrap, and Git version control.

I am proficient in using:

  • Html and Css
  • Bootstrap

  • JQuery
  • JSON and AJAX
  • Node.js
  • Databases like:

  • Git version control
